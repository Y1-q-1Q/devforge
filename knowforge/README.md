# KnowForge

> Knowledge management system for developers

## Quick Start

```bash
# Install dependencies
npm install

# Install Rust dependencies
cd src-tauri
cargo build

# Run development
npm run dev

# Build release
npm run build
```

## Tech Stack

- **Frontend**: React + TailwindCSS
- **Backend**: Rust + Tauri
- **Database**: SQLite
- **AI**: Local LLM support

## Features

- [x] Note management
- [x] Full-text search
- [x] Tag system
- [ ] AI auto-tagging
- [ ] Browser extension
