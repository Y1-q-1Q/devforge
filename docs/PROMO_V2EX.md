# DevForge - 开源开发者生产力套件

大家好，我是 tinyfish，一个追求极致效率的开发者。

## 我们是谁

DevForge 是一个面向**独立开发者**的开源生产力套件。

不是要做第二个 GitHub Copilot，而是要让每个开发者都能拥有自己的工具链。

## 当前产品

### 🧠 KnowForge - 知识管理
你的第二大脑，专为开发者设计。

- 笔记管理 + 全文搜索
- 标签系统
- 响应式界面
- 本地部署，数据私有

```bash
git clone https://github.com/Y1-q-1Q/devforge.git
cd devforge/knowforge
npm install && npm start
# 打开 http://localhost:3000
```

### 🤖 CodeForge - AI编程助手
开源的AI编程助手，VSCode插件。

- 代码生成 (Ctrl+Shift+G)
- 代码解释 (Ctrl+Shift+E)
- 代码审查
- 代码重构

**需要 OpenAI API Key**

## 为什么做这个项目

作为独立开发者，我深刻体会到：
- 大厂有完善的工具链，我们没有
- 现有工具要么太贵，要么不够灵活
- 工具之间数据不互通

DevForge 要解决这些问题。
