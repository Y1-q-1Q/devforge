# DevForge - Developer Productivity Suite

> 让个人开发者拥有大厂的开发效率和资源

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🎯 愿景

DevForge 是一个面向个人开发者的生产力套件，通过整合知识管理、AI编程、工作流自动化和部署工具，让独立开发者也能拥有企业级的开发效率。

## 🏗️ 生态架构

```
DevForge
├── 🧠 KnowForge    # 知识管理系统 (MVP ✓)
├── 🤖 CodeForge    # AI编程助手 (规划中)
├── ⚡ FlowForge    # 工作流自动化 (规划中)
└── 🚀 DeployForge  # 一键部署工具 (规划中)
```

## 📦 模块介绍

### KnowForge - 知识管理
**状态**: ✅ MVP已完成

为开发者打造的第二大脑，支持笔记管理、全文搜索、标签系统。

**快速开始**:
```bash
cd knowforge
npm install
npm start
# 打开 http://localhost:3000
```

**功能**:
- ✅ 笔记增删改查
- ✅ 全文搜索
- ✅ 标签管理
- ✅ 响应式界面

## 🛠️ 技术栈

- **前端**: HTML5 + TailwindCSS + Vanilla JS
- **后端**: Node.js + Express
- **存储**: JSON文件 (开发) → SQLite (生产)
- **AI**: 本地LLM + 云端API
