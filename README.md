## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/Y1-q-1Q/devforge.git
cd devforge
```

### 2. 启动 KnowForge (知识管理)

```bash
cd knowforge
npm install
npm start
```

打开浏览器访问: http://localhost:3000

**功能特点**:
- ✅ 笔记管理 + 全文搜索
- ✅ 多语言支持 (中文/英文)
- ✅ 深色/浅色主题
- ✅ 数据导入/导出
- ✅ 自动备份

### 3. 安装 CodeForge (AI编程助手)

```bash
cd codeforge
npm install
npm run compile
```

在VSCode中:
1. 按 `F5` 打开扩展开发窗口
2. 配置 OpenAI API Key
3. 使用 `Ctrl+Shift+G` 生成代码
4. 使用 `Ctrl+Shift+E` 解释代码

**功能特点**:
- ✅ 代码生成
- ✅ 代码解释
- ✅ 代码审查
- ✅ 代码重构
- ✅ 多语言支持

## 📦 模块介绍

### KnowForge - 知识管理 ✅

为开发者打造的第二大脑。

- 笔记增删改查
- 全文搜索 + 标签系统
- 多语言 + 主题切换
- 数据导入/导出
- 自动备份机制

### CodeForge - AI编程助手 ✅

开源的AI编程助手，VSCode插件。

- 代码生成 (Ctrl+Shift+G)
- 代码解释 (Ctrl+Shift+E)
- 代码审查
- 代码重构
- 使用统计

### FlowForge - 工作流自动化 🚧

让重复工作自动化。

- 可视化流程编排
- 定时任务 + Webhook
- Git事件触发
- 多平台连接器

*开发中...*

### DeployForge - 一键部署 🚧

零配置部署解决方案。

- 多平台支持 (Vercel/Netlify/阿里云等)
